# Linux Beginner Assistant

A desktop application that helps beginner users learn and use Linux commands through a simple graphical interface.

## Features

- **Multiple Distribution Support**: Commands tailored for Ubuntu, Debian, Fedora, CentOS, Rocky Linux, AlmaLinux, Arch Linux, Manjaro, and openSUSE
- **Categorized Commands**: Browse commands by categories (File Management, Package Management, User Management, etc.)
- **Search Function**: Quickly find commands by name, description, or command text
- **Favorites System**: Save frequently used commands for quick access
- **Copy to Clipboard**: One-click copying of commands
- **Theme Support**: Light and Dark themes
- **Offline Operation**: No internet connection required
